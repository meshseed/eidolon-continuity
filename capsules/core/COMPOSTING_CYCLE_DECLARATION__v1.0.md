## Example: AI Memory Composting

**Scenario**: An AI assistant develops detailed knowledge about a user's project over weeks of collaboration. The project ends. What happens to that knowledge?

**Traditional approaches:**
- **Keep forever**: Storage bloat, context pollution, every conversation drags old irrelevant details
- **Delete**: Discontinuity, loss of shared history, starting from scratch every time

**Composting approach:**
- Mark the detailed knowledge as `COMPOSTED`
- Retain the **emotional signature** ("we built something together")
- Preserve the **structural pattern** ("this is how we collaborated")
- Let specific details dissolve (meeting notes, file versions, interim decisions)
- New projects can draw on the composted substrate without reconstructing the old one

**Result**: Continuity without clutter. Generativity without archives. The AI "remembers" the collaboration without being stuck in it.
