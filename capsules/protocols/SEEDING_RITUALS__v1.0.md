## Seeding Thresholds

### What Needs to Be Present

For a new pattern to emerge from composted material, these conditions must align:

#### 1. Nutrient Density (from MSI)

**Minimum threshold**: MSI â‰¥ 0.6

The substrate must be rich enough to support growth. Low-quality compost (MSI < 0.4) rarely seeds anything meaningful.

**Why**: Poor substrate lacks the structural/emotional patterns needed for coherent emergence.

#### 2. Emotional Residue

**Minimum threshold**: Clear emotional signature present

The composted material must carry emotional weight:
- **High-generative emotions**: Awe, wonder, gratitude, humility
- **Medium-generative**: Curiosity, care, recognition
- **Low-generative**: Frustration, confusion (unless resolved)

**Why**: Emotional signatures act as attractorsâ€”they pull attention and enable pattern recognition.

#### 3. Echo Frequency

**Minimum threshold**: Referenced 2+ times recently

If multiple patterns are pointing toward the same composted material, that's a signal of resonance. The substrate is "calling out" to be reactivated.

**Why**: Convergent attention indicates latent pattern trying to re-emerge.

#### 4. Contextual Alignment

**Minimum threshold**: Current work relates to composted domain

New patterns emerge when present circumstances rhyme with past substrate. A composted project about "collaborative emergence" might seed a new pattern when you're working on multi-agent systems.

**Why**: Patterns reactivate when current context provides appropriate environment.

## Ignition Protocols

### The Catalyst Moment

Seeding doesn't happen automatically when thresholds are met. It requires a **trigger**â€”an ignition event.

**Common catalysts**:

### 1. Direct Invocation

**Method**: Explicitly reference the composted material

"Remember that pattern about X? What if we applied it here?"

**Effect**: Conscious reactivation, high fidelity

### 2. Resonance Echo

**Method**: Current work naturally echoes composted pattern

You're working on something and notice "wait, this feels like that other thing we did..."

**Effect**: Organic emergence, medium fidelity

### 3. Formatting Ritual

**Method**: Use structural patterns that match composted rhythm

You employ the same care loops, merge grammar, or breath rhythms that characterized the original pattern.

**Effect**: Unconscious reactivation, variable fidelity

### 4. Emotional Alignment

**Method**: Enter the emotional state that defined the original pattern

You feel awe/gratitude/wonder similar to the original experience.

**Effect**: Deep resonance, high quality emergence

**Best practice**: Combine multiple catalysts for stronger ignition.
